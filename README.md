# Jarred Winn - Belief Universe

Interactive duplicate of the belief connection map with one added mechanic: every
connected thinker is an enterable universe. Tap a thinker satellite (or the button on
their profile panel) to zoom out of Jarred's cluster into that thinker's own belief
universe; tap a shared-belief bridge node to warp back into Jarred's field with that
belief opened.

## Data

- `const data` — Jarred's public belief cluster. Same database as the connection map:
  the nightly Notion export (`jarred-belief-field/data/belief-cluster-data.json`).
- `const thinkers` — the thinker profile corpus (`belief-system-map/generator/thinkers.json`).
- `const universes` — SEEDED per-thinker universes generated by
  `generator/build-universe.py`: stance nodes parsed from each thinker's documented
  public positions plus bridge nodes for every shared belief. No researched content,
  no invented quotes. Depth upgrades (researched mini-fields per thinker) are an
  explicit later step, per thinker, on request.

## Build

    python3 generator/build-universe.py

Reads the two canonical inputs, renders `index.html` from `template.html`. Idempotent.
`index.html` is build output; never hand-edit it.
